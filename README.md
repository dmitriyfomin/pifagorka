# Задание для сайта pifagorka
Сверстать страницу с заголовком «Подписка на рассылку». Ниже две колонки. В левой колонке текст: «Чтобы быть в курсе последних событий в жизни Пифагорки, подпишитесь на нашу рассылку. Спамить не будем.» В правой — форма с двумя полями: имя и адрес эл. почты, и кнопкой «Подписаться». Все поля формы обязательны для заполнения. Отправка формы происходит без перезагрузки страницы. После отправки на месте формы появляется текст: «Вы подписались на рассылку. Спасибо!» Данные из формы попадают в базу данных mysql. Таблицу сконструировать самостоятельно, дамб выслать в архиве вместе с результатом задания на почту. Максимальная ширина страницы 940 px. На экранах телефонов (568 px и меньше) колонки с текстом становятся одна над другой. Для вёрски страницы необходимо использовать фреймворк PureCSS (http://purecss.io/).Для работы с базой данных — библиотеку doBD (https://github.com/vasa-c/go-db).
